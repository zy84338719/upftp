## 🔒 安全改进

### Before (之前)
```go
// API 端点 - 未保护
mux.HandleFunc("/api/info", HandleServerInfo)      // ❌ 未保护
mux.HandleFunc("/api/tree", HandleDirectoryTree)  // ❌ 未保护
mux.HandleFunc("/api/upload", handleUpload)        // ❌ 未保护

// 静态文件 - 未保护
mux.Handle("/files/", http.FileServer(...))        // ❌ 未保护
```

### After (之后)
```go
// API 端点 - 已保护
mux.HandleFunc("/api/info", withAuth(HandleServerInfo))      // ✅ 已保护
mux.HandleFunc("/api/tree", withAuth(HandleDirectoryTree))  // ✅ 已保护
mux.HandleFunc("/api/upload", withAuth(handleUpload))        // ✅ 已保护

// 静态文件 - 已保护
mux.HandleFunc("/files/", withAuth(handleFiles))             // ✅ 已保护
```

## 🚀 使用方法

### 方式 1: 命令行参数
```bash
# 快速启用认证
./upftp -http-auth -http-user admin -http-pass yourpassword

# 只启用认证，使用默认用户名密码
./upftp -http-auth
```

### 方式 2: 配置文件
```yaml
# upftp.yaml
http_auth:
  enabled: true
  username: "admin"
  password: "yourpassword"
```

```bash
# 使用配置文件
./upftp
```
